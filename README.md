## 📚 Conteúdo por Tema

### Hello World
Primeiro contato com Python, usando a função `print()` para exibir uma mensagem na tela.

### Tipos de Dados
Exploração dos tipos primitivos do Python:
- `int` — números inteiros
- `float` — números decimais
- `bool` — valores `True` e `False`
- `str` — texto (strings)

### Variáveis e Constantes
Como declarar variáveis, reatribuir valores e realizar atribuição múltipla em uma única linha. Convenção de nomes para constantes em `SNAKE_CASE_MAIÚSCULO`.

### Conversão de Tipos
Uso das funções de conversão para transformar dados entre diferentes tipos, como converter string para número ou número para string.

### Entrada e Saída de Dados
Uso das funções `input()` para capturar dados do usuário e `print()` para exibir resultados formatados. Inclui o uso de `type()` para verificar o tipo de uma variável.

### Concatenação de Strings
Duas formas de combinar strings e variáveis:
- Usando o operador `+`
- Usando **f-strings** (forma moderna e recomendada)
